# 🧑‍💼 Sistema de Gerenciamento de Funcionários

![Python](https://img.shields.io/badge/Python-3.x-blue)
![PyQt5](https://img.shields.io/badge/PyQt5-GUI-green)
![SQLite](https://img.shields.io/badge/SQLite-Database-lightgrey)
![Status](https://img.shields.io/badge/status-em%20desenvolvimento-yellow)

Aplicação desktop desenvolvida em **Python** utilizando **PyQt5**, com o objetivo de gerenciar funcionários de forma simples e eficiente.

---

## 🚀 Funcionalidades

- ✅ Cadastro de funcionários  
- 🔍 Busca por matrícula  
- 🗑️ Remoção de funcionários  
- 📊 Contagem total de funcionários ativos  
- 📥 Exportação dos dados em CSV  
- 🎨 Interface gráfica moderna com QSS  
- 🖥️ Navegação por múltiplas páginas  

---

## 🛠️ Tecnologias utilizadas

- Python 3  
- PyQt5  
- SQLite3  
- CSV (exportação de dados)  

---

## 💾 Banco de dados

O banco de dados é criado automaticamente ao iniciar o sistema.

> ⚠️ O arquivo `.db` não é incluído no repositório por boas práticas.

---

## 📥 Exportação de dados

O sistema permite exportar todos os funcionários para um arquivo `.csv`, compatível com Excel e Google Sheets.

---

## 🎨 Interface

A interface foi construída com PyQt5 e estilizada utilizando QSS, oferecendo:

- 🌙 Dark mode 
- 📐 Layout organizado  
- ✨ Experiência mais agradável
